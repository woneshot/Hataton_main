# Hackathon Project: Award-Winning Style & Sound

This repository contains the source code for a game developed during a **48-hour Game Development Hackathon**. Our team was honored to win the **"Best Art & Sound Style"** award for this project.

## 🏆 Achievements:
- **Award:** Best Visual & Audio Aesthetic.
- **Role:** Team Lead & Sound Designer / Developer.
- **Challenge:** Developed a playable build from scratch within a strict 48-hour deadline.

## 🕹 Project Highlights:
- **Atmospheric Soundscape:** Custom-crafted audio designed to enhance the game's unique dark aesthetic.
- **Visual Polish:** Integration of 2D/3D assets with specific post-processing to achieve a cohesive art style.
- **Team Collaboration:** Successfully led a multi-disciplinary team, managing tasks and priorities in a high-pressure environment.

## 🛠 Tech Stack:
- **Engine:** Godot
- **Audio:** Ableton Live (Sound Design & Production)
- **Version Control:** Git
